####
Datasets from McGowen et. al. (2020) can be accessed at Dryad at https://datadryad.org/dataset/doi:10.5061/dryad.jq40b0f.

Download the "DATASET_A.phylip.txt" file from Dryad, and upload to the “McGowen_alignments” directory. Run the provided iqtree script to generate a concatenated empirical phylogeny.
