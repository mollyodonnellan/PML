####
Dataset for Wickett et. al. (2014) available at https://datacommons.cyverse.org/browse/iplant/home/shared/onekp_pilot/PNAS_alignments_trees/gene_alignments/filtered/FNA2AA

Following the link, navigate under the “filtered” directory, select the “filtered_FNA2AA_alignments.tar.gz” file, and download. 

Download the dataset and filter to retain only those files which end in *.c1c2.f25. This can be done by filtering using CTRL+F, and searching “.c1c2.f25” in quotations. Move these files to a separate folder, and name it “FNA2AA_Filtered”. Upload the folder into the “Wickett_alignments” directory. 

Run the accompanying iqtree shell script to generate a concatenated empirical phylogeny.
