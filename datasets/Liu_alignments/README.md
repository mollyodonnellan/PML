####
Dataset from Liu et. al. 2017 available at https://datadryad.org/stash/dataset/doi:10.5061/dryad.bp462

Download the "seq.zip" zipfile from Dryad, unzip onto your desktop.

Go into the file and delete the folder labeled “p3” and “p12” and leave just “cds”.

Upload the “cds” file into the “Liu_alignments” directory.

Run the accompanying iqtree shell script to generate a concatenated empirical phylogeny.
